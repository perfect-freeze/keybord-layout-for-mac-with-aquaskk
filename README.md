keybord-layout-for-mac-with-aquaskk
===================================

mac with AquaSKK 用 月配列 俺式

インストール
------------

1. このリポジトリをクローン
2. `~/Library/Application Support/AquaSKK` に keymap.conf と kana-rule.conf をコピー
3. 下記レイアウトをプリントアウトして、席の右側のパーティションに画鋲で貼り付ける

キーバインディング
------------------
